Payment ID Generator

Press the button to generate random PaymentID or enter Payment ID to decode it

Enter short memo to encode it (ASCII only)

Payment ID is optional transaction attachment that is usually used to identify transactions. Because of anonymity of Karbowanec transactions, without Payment ID you will not be able to distinguish them. You can generate random Payment IDs and give them to each user who is to send you some Karbo. This way you can tie payments to users.
